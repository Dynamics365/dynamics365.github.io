These methods are commonly used while performing a calculation or to lookup fields from another table. The 

Display modifier is added before the method name that shows that the methods return value is to be displayed on forms and reports.

ex:

  1. display NumberOfRecords testMethod(CustTable myTable)
  2. {
  3.    NumberOfRecords recCount = this.totalNumberOfRows();
  4.    Return recCount;
  5. }


Please note that the return type of a display method should be of type EDT, in case is Number of Records.

[more]

Usage:

The Display modifier can be used on :

  • Table methods.
  • Form methods.
  • Form datasource methods.
  • Report methods.
  • Report design methods.
Avoid complex calculations on display methods because they are called each time a form is redrawn.

Add a display method to lookup fields from another table on a form datasource.

This is how the method looks.

  1. display WorkItemType displayWorkItemType()
  2. {
  3.     WorkflowWorkItemTable workflowWorkItemTable;
  4.     WorkflowWorkItemQueue workflowWorkItemQueue;
  5.  
  6.     select firstonly Queue from workflowWorkItemTable
  7.         Where workflowWorkItemTable.RefRecId == custInvoiceTable.RecId
  8.             && workflowWorkItemTable.RefTableId == tableNum(CustInvoiceTable)
  9.         join WorkitemType from workflowWorkItemQueue
  10.             where workflowWorkItemQueue.RecId == workflowWorkItemTable.Queue;
  11.  
  12.     return workflowWorkItemQueue.WorkitemType;
  13. }

As , you can see the methods lookup WorkItemType from another table that is'nt available under the datasource node of this form.
