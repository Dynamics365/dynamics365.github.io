This blog post is show how to apply OR conditions in query build ranges in a simple way on same field in a table.

Let's see the simple query :

select * from CustTable where AccountNum == '2001' || AccountNum == '2002'

We can find out solutions on MSDN Â by using expression in query ranges, but as it has lot of specifications which needs to be followed.

However there is a simple way to do it :

CustTable cust;
Query query = new Query();
QueryBuildDataSource qbds;
QueryBuildRange queryRange1, queryRange2;
;

qbds = query.addDataSource(tableNum(CustTable));

queryRange1 = qbds.addRange(fieldNum(CustTable, AccountNum));
queryRange1.value('2001');

queryRange2 = qbds.addRange(fieldNum(CustTable, AccountNum));
queryRange2.value('2002');

info(qbds.toString());
